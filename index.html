🔒 Privacy & Data Notice

This tool is designed with data privacy as a core principle:

  • All data stays in your browser. No CSV data is uploaded to any server, database, or cloud storage.
  • Nothing is stored. When you close this tab, all uploaded data is permanently gone. No cookies, no local storage, no saved files.
  • Postcode lookups are anonymous. Only postcodes are sent to Postcodes.io (a free UK government-backed service) to get coordinates. No names, roles, or personal data is transmitted.
  • Place name lookups are anonymous. Only town/city names are sent to get approximate coordinates. No personal details included.
  • The host (GitHub Pages) never sees your data. It only serves the code file.
  • No analytics or tracking. No Google Analytics, no cookies, no tracking scripts of any kind.

For questions about data handling, contact your programme manager.
